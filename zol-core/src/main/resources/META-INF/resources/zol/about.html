Zol is a log viewer (a tail for the web) that can be deployed to Java Servlet Containers
Zol uses
java.util.logging
to add a custom Handler that stream log records to a
web socket
connection.
This allows you to view the log online. You can also change the log level of your logger on the fly.
All log levels are restored when the log viewer are stopped.
